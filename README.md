Data Analysis Projects
Overview
This repository contains my personal data analysis projects created using Excel, Power BI, Tableau, and Python.
The projects focus on analyzing business and sales data and presenting insights through dashboards and visualizations.

Repository Structure
Excel/ – Excel dashboards and analysis files
PowerBI/ – Power BI reports and dashboards
Tableau/ – Tableau dashboards
Python/ – Python scripts for data analysis and visualization
Data/ – Raw data files (CSV, Excel, TXT, JSON)
README.md – Project description and documentation

Notes
All projects are created for learning and practice purposes.
Each tool folder contains the relevant analysis files and dashboards.
Visualizations aim to demonstrate data insights and reporting skills.
